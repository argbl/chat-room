# 项目说明

项目前端采用vue3+typescript+pinia，

后端使用egg.js搭建。 

主要使用socket.io和egg-socket.io实现聊天的核心功能。 

页面设计上参考国外网站discord。

## 项目后端地址

[Martinell1/record: 左边那个项目的后端，采用egg.js实现 (github.com)](https://github.com/Martinell1/record)
